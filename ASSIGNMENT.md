# Back-end Developer Assignment

Welcome to the ANSCER Robotics Back-end Developer assignment! We appreciate your interest in joining our team. This assignment is designed to assess your skills in backend development, particularly with Node.js, Express.js, MongoDB, and other relevant technologies.

## Assignment Details

### Task


### Requirements

- Use Node.js and Express.js to build the API.
- Use MongoDB as the database.
- Implement the API endpoints for creating, reading, updating, and deleting tasks.
- Use TypeScript for type safety.
- Implement basic validation for the input data.
- Document the API endpoints using a tool like Postman or Swagger.

### Bonus Points

- Implement user authentication and authorization.
- Use Docker to containerize the application.
- Write unit tests for the API endpoints.
- Provide a Postman collection or Swagger documentation for testing the API.

### Submission

1. Fork this repository to your own GitHub account.
2. Implement your solution in your forked repository.
3. Once you have completed the assignment, create a pull request to this repository with your solution.
4. Include any additional documentation or instructions needed to run your application in the `README.md` file of your forked repository.

We look forward to seeing your work! If you have any questions, please feel free to reach out.

Good luck!
